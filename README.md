# Trancelate-Service-Client
Java Swing, Java Web, XML

This is an application that translates from Serbian to English and from English to Serbian.

  1. Run service
  2. Run client
  3. Trancelate
  
Available words:

  -Serbian:
    1. prijatelj
    2. trag
    3. tajna
    4. rodjak
    5. razgovor
    
  -English:
    1. friend
    2. trail
    3. the secret
    4. cousin
    5. conversation
